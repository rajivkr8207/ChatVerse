# 🚀 ChatVerse

ChatVerse is a modern AI-powered real-time chat application built with React, Node.js, Socket.IO, MongoDB, and LangChain. It delivers intelligent AI conversations, real-time messaging, authentication, and a sleek user experience.

![ChatVerse Banner](https://ik.imagekit.io/rjdev/usingpic/image.png)

---

## ✨ Features

- 🤖 AI Chat Assistant
- ⚡ Real-Time Messaging with Socket.IO
- 🧠 LangChain Integration
- 💬 Context-Aware Conversations
- 🔐 JWT Authentication
- 🌐 Google OAuth Login
- 📱 Fully Responsive Design
- 🌙 Dark Mode Support
- 📜 Chat History
- 🔥 Trending Topic Suggestions
- 🛡️ Secure Backend Architecture
- 🚀 High Performance & Optimized UI

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/rajivkr8207/ChatVerse
```

### Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

### Backend Setup

```bash
cd backend

npm install

npm run dev
```
